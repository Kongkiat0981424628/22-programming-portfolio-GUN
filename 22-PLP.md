28/11/68
เป้าหมาย-แก้ไขปัญหาวิ่งโดยไม่ทราบข้อมูลในการวิ่งออกกำลังกาย
ปัญหาที่สนใจ-วิ่งโดยไม่ทราบข้อมูลในการวิ่งออกกำลังกาย (pace)

12/12/68 ขั้นตอนของโปรแกรมแก้ไขปัญหาวิ่งโดยไม่ทราบข้อมูลในการวิ่งออกกำลังกาย
วิ่งโดยวัดระยะทาง
1.ตั้งแจ้งเตือนเมื่อวิ่งได้ pace ที่ต้องการ
2.ทำฟีเจอร์ที่หลากหลายมากขึ้น เช่น คำนวณ calorie
ทำเป็น GUI
3.ทดลองโปรแกรม
4.ปรับปรุงและแก้ไขตามข้อผิดผลาด
6.Iput คือ การวัด pace ในการวิ่งที่ได้
7.Process คือ เวลา/ระยะทางที่ได้
8.Ourput คือ การแจ้งเตือนเมื่อวิ่งได้ pace ที่ต้องการ โจทย์ที่1 ไม่ยากเลยครับสำหรับกัน โจทย์ที่2 ชอบผังงานมากกว่าเพราะเข้าใจได้ง่ายมากกว่า โจทย์ที่3 ขั้นตอนการเขียนโปรแกรมครับผม

19/12/68 ทำงานกลางภาคโดยใช้ดจทย์ที่ได้รับคือ ขนาดไฟล์:รับขนาดไฟล์ถ้ามากกว่า 25 MB ให้บอก"ไม่สามารถส่ง Gmail ได้" โดยให้เขียน input ourput process และ บอกเงื่อนไขที่เป็นจริง และ ไม่เป็นจริงพร้อมเขียนโปรแกรมโค้ดใน Colab

20/1/69 เขียนโปรแกรมในโค้ด colab และแก้ไขหน้า GUI ใน canva โดยใช้หัวข้อคือ การคำนวณ pace ในการวิ่ง Calculate run pace

23/1/69 แก้ไขโค้ดโปรแกรมให้สมบูณ์และจับคู่เพื่อแสดงความคิดเห็นใน github
